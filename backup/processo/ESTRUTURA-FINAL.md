## 📋 INTEGRAÇÃO GITHUB COPILOT

**Onde Copilot entra:**

1. **Você preenche:** [prompts/relatorio-simples.prompt.md](prompts/relatorio-simples.prompt.md)
2. **Você copia:** Tudo até seção "PROMPT PARA O COPILOT"
3. **Você cola:** No Copilot Chat (Ctrl+I)
4. **Copilot segue:** Regras em [.github/copilot-instructions.md](.github/copilot-instructions.md)
5. **Copilot gera:** JRXML parametrizado + safe
6. **Você valida:** `npm run validate` (scripts/validate.js)
7. **Você compila:** `npm run compile` (scripts/compile.js)
8. **Resultado:** .jasper + .pdf + metadata.json
